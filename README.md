## 💡 Overview

<a href="#postico">Postico</a> serves developers who work with PostgreSQL databases and want a client that matches the design quality they expect from Mac software — not a Java-based cross-platform tool with platform-generic UI, but a native Mac application that feels at home on macOS.

The <a href="#postico">developer database exploration use case</a> is most common — a backend developer checking what data is in the database during development, verifying that migrations applied correctly, checking the structure of tables, and running diagnostic queries. Postico makes this exploratory database work fast and low-friction with its visual table browser and quick SQL editor. The <a href="#postico">data analyst workflow</a> uses Postico's SQL editor and export capabilities for ad-hoc data analysis — writing queries to extract and aggregate data, exporting results for further analysis, and exploring database contents without requiring a full analytics platform.

The <a href="#postico">database administrator use case</a> uses Postico for structure management, connection management, and administrative operations on PostgreSQL databases. <a href="#postico">Eggerapps development quality</a> from a small independent Mac developer with a track record of high-quality, thoughtfully designed Mac database tools — Postico has established Eggerapps as a respected name in Mac database tooling.

<a href="#postico">Dark mode support</a> follows the macOS system appearance — Postico in dark mode provides a comfortable database working environment that matches the dark mode desktop many developers prefer. <a href="#postico">Multiple simultaneous connections</a> open connections to different databases in separate windows — working with development and staging databases simultaneously without connection switching.

<a href="#postico">SSH tunnel support</a> connects to databases through SSH tunnels — accessing databases on remote servers that aren't directly accessible on the network, connecting to cloud database instances that require SSH for security. <a href="#postico">Continuous Postico 2 development</a> by Eggerapps delivers ongoing improvements with new features, PostgreSQL version compatibility, and macOS version support maintained through active development.

## 💬 FAQ

<details>
<summary>How does Postico compare to TablePlus?</summary>

Both are native Mac database clients. Postico focuses exclusively on PostgreSQL with deep Postgres-specific support; TablePlus supports multiple database systems. Postico's PostgreSQL-specific focus provides a more tailored experience for Postgres-only workflows.

</details>

<details>
<summary>Does Postico support PostgreSQL views and functions?</summary>

Yes. The database structure browser shows tables, views, functions, sequences, indexes, and other database objects.

</details>

<details>
<summary>Is there a free trial for Postico?</summary>

Check the Postico website for current trial availability and licensing options.

</details>

<details>
<summary>Does Postico run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>

<details>
<summary>Can Postico filter table data?</summary>

Yes. Column value filtering and sorting are available in the table content view for navigating large tables without writing SQL.

</details>

<details>
<summary>What PostgreSQL versions does Postico support?</summary>

Postico supports modern PostgreSQL versions. Check current Postico documentation for specific version compatibility.

</details>
